Airbnjh93

Teams and Code Reviews

This project is an Airbnb clone featuring a booking and review system, created by a team of 4 during the Le Wagon bootcamp. It was my first time working on a team larger than two people, we worked in branches and reviewed eachothers pull requests. We used mob sessions and pair programming to implement and solve some of the harder features and bugs. Since the bootcamp I've forked the repository and started making tweaks, refactoring, and implementing features that we didn't have time to finish during the bootcamp. Visit Airbnjh93

Frontend
