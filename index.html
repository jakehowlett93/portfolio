Pythia

Empowering People

A website that allows non-tech users to explore and visualize their databse without needing to learn SQL or hire an analyst. This project was built as a team of 4, using GitHub for version control, and GitHub issues for ticket tracking. We used the rails framework alongside HTML, SCSS and JS to build the app.

Backend

I focused almost exclusively on the backend for this project, I worked on the seed file, generated some of the models and controllers then adding their validations and methods. My favourite part was building the logic that turns the query form into an actual SQL query. The app uses a postgres database hosted on Heroku.
